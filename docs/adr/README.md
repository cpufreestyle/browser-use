# Architecture Decision Records

本目录存放项目的架构决策记录（ADR）。

每个 ADR 文件按编号命名，格式为 `NNNN-<slug>.md`（如 `0001-use-playwright-for-browser-automation.md`）。

## ADR 模板

```markdown
# NNNN. <决策标题>

日期: YYYY-MM-DD

## 状态

已接受 / 已废弃 / 已替代（被 ADR-NNNN 替代）

## 背景

<描述做出此决策的背景和上下文>

## 决策

<描述做出的决策>

## 后果

<描述此决策带来的正面和负面后果>
```
